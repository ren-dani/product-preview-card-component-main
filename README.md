### What I learned
How to center a div vertically and horizontally.

```css

/*Using grid*/

body {
    background: var(--Cream);
    /* XY centering grid */
    min-height: 100vh;
    display: grid;
    place-items: center;
}


/*Using flexbox*/
body {
	display: flex;
	align-items: center;
	justify-content: center;
}

```
